スケルトン
Skeleton

概要
====

Keynote 向けのボーンポリゴンの作成を支援します。
これは Metasequoia 用の Command プラグインです。

このプラグインを使用するには .NET Framework 4.0 以上が必要です。

インストール
===========

プラグインフォルダの中の Command フォルダに Skeleton.dll を置いてください。
すでに旧バージョンがインストールされている場合は、[プラグインについて] ダイアログから上書きすることはできません。手動で入れ替えてください。

使い方
=====

ボーン モード
------------

・ボーンの始点としたい点を画面上でクリックし、その後ボーンの終点としたい点をクリックするとボーンポリゴンができます。
・続いてクリックすると結合されたボーンを次々に作成できます。
・右クリックで始点指定を解除できます。
・始点を指定する際、既存のボーンの先の頂点を選択すると、そこに結合したボーンを作成できます。
・終点を指定する際、通常ボーンの作成であればクリックしたままドラッグすることによりボーンポリゴンの方向、およびサイズを変化させることができます。
　(浮動ボーンはサイズを指定できません)
・終点は既存の頂点の位置を指定できますが、既存の頂点には結合されません。
・Metasequoia のグリッドスナップが有効な場合、スナップします。
・[ボーン名] は生成する材質の名前を指定できます。
　特定の名前の場合、生成後次の名前が自動的にセットされます。
　数字で終わる名前の場合、生成後次の名前は連番になります。

アンカー モード
-------------

・選択された材質で、選択された頂点を囲む、選択されたオブジェクトに関連するアンカーを作成します。
　生成されるアンカーはただのボックスですので、必要に応じて加工してください。
・画面上をドラッグすることにより現在のオブジェクトの頂点を範囲選択できます。
・[ボーンの軸に沿う] が有効の場合、ボーンの傾きに沿って傾いたボックスを作成します。
　ただし傾きの方向によってはボックスのサイズがとても大きくなる可能性もあります。
・[対称アンカーの自動作成] が有効の場合、材質名が [] で終わる場合
　生成されるアンカーのオブジェクトを自動的に X 鏡面設定します。
